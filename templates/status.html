API Status

If you can view this page, the API is running properly.

Endpoints

GET:

  • / : View this page.
  • /rooms : Get the list of rooms.
  • /backup : Download the database backup.
  • /bookings : Get the list of bookings.
  • /bookings/<email> : Get the list of bookings by specific email. (<email>'s @ need to be encoded as %40')

POST:

  • /check : Check availability of rooms. (type: int, amount: int, checkin: str, checkout: str)
  • /book : Book available rooms. (email: str, name: str, type: int, amount: int, checkin: str, checkout: str)
